Looking for a way to apply myself as a first post-collegiate pursuit, I began to learn the Java programming language professionally. And, not knowing all of the aspects, I stumbled my way into Object-oriented programming as a programming paradigm.
I was nervous. I was afraid. I did not know what I was doing. And, I did not know how things were going to work out. On top of those feelings, I had to learn how to program.
As a result, I would either fail and give up or learn from each failure and try to project myself forward. I learned to learn in college, but as a Java Developer I learned how to learn quickly. The Java language assisted me, but learning programming under pressure was more valuable.
I began to discover that I could focus my attention precisely, and it motivated me to be more specific in arenas outside of programming. I learned that I could build software for others.
Figuring out that software was an ultimate tool of the era we live in allowed me to begin the shaping of other exciting new tools. But, I now try to apply my programming skills towards new kinds of programming such as Machine Learning and Affective Computing.
