I never really thought that I was starting down a path of software development when I wanted to learn HTML. I thought CSS was very similar to that of an InDesign rule for page layout. And, I though JavaScript was a way of making the HTML dance like an animation in Adobe Flash or After Effects. When I decided to learn Java, I also decided to rethink some of my ignorance towards user interface development.
I felt amazed, and shocked. But, I also felt content.
I learned that a user interface becomes the portal for humans. Learning about the future of augmented reality and virtual reality challenges this role. It means that I might be able to assist in redefining what the front end really is in a computer system. In the before times, it stopped at the terminal screen, but now the front of a system might be one (or many) of many screens, devices, or some other interface.
My excitement towards how humans use computers will never cease.
I'm thrilled to see people use computers, and I hope that computers and humans become a lot friendlier towards each other in the future.
